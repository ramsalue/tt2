\subsubsection{Diseño de la interfaz humano-máquina (HMI)}
La interfaz humano-máquina inició con el diseño de las diferentes pantallas que el usuario observará durante el uso del sistema. En este aparatado se describe cada una de las pantallas a través de las cuales el usuario, en este caso destinado al fisioterapeuta, interactuará con el sistema.\\
Al iniciar, la primera pantalla que se presenta solicita calibrar el sistema, en la Fig. \ref{fig:AppV1_Calibracion} se muestra la calibración del sistema visible para el usuario.
\begin{figure}[h!]
	\centering
	\includegraphics[width=0.6\textwidth]{figure/pantallas_interfaz/App Ver1.0/Calibracion.png}
	\caption{Calibrando sistema.\label{fig:AppV1_Calibracion}}
\end{figure}

Una vez que el sistema ha sido calibrado permite a través de la pantalla mostrada en la Fig. \ref{fig:AppV1_Calibrado} utilizar el botón "Comenzar Sesión". 
\begin{figure}[h!]
	\centering
	\includegraphics[width=0.6\textwidth]{figure/pantallas_interfaz/App Ver1.0/Calibrado.png}
	\caption{Sistema calibrado.\label{fig:AppV1_Calibrado}}
\end{figure}

El usuario tendrá la opción de elegir el tipo de rehabilitación [Fig. \ref{fig:AppV1_Seleccion}] para posteriormente proporcionar parámetros útiles para el funcionamiento del sistema, tales como el número de repeticiones [Fig. \ref{fig:AppV1_Movimientos}]. 
\begin{figure}[h!]
\centering
\includegraphics[width=0.6\textwidth]{figure/pantallas_interfaz/App Ver1.0/Seleccion.png}
\caption{Selección del tipo de rehabilitación a realizar.\label{fig:AppV1_Seleccion}}
\end{figure}
\begin{figure}[h!]
	\centering
	\subcaptionbox{Configuración parámetros para movimiento de abducción-aducción.\label{fig:AppV1_Abduccion_Aduccion}}
	{\includegraphics [trim = 0 0cm 0 0, clip,width=0.6\textwidth]{figure/pantallas_interfaz/App Ver1.0/Abduccion_Aduccion.png}}
	\subcaptionbox{Configuración parámetros para movimiento de flexión-extensión.\label{fig:AppV1_Flexion_Extension}}
	{\includegraphics [trim = 0 0cm 0 0, clip,width=0.6\textwidth]{figure/pantallas_interfaz/App Ver1.0/Flexion_Extension.png}}
	\caption{Configuración de parámetros para movimientos del sistema.\label{fig:AppV1_Movimientos}}
\end{figure}

Una vez seleccionado el movimiento a realizar y configurado sus parámetros, se muestra la pantalla de inicio de la Fig. \ref{fig:AppV1_Inicio} con la cual el usuario podrá iniciar la sesión de rehabilitación.
\begin{figure}[h!]
	\centering
	\includegraphics[width=0.6\textwidth]{figure/pantallas_interfaz/App Ver1.0/Inicio.png}
	\caption{Pantalla de inicio de sesión de rehabilitación.\label{fig:AppV1_Inicio}}
\end{figure}

Durante el proceso de funcionamiento del sistema se mostrará al usuario la pantalla con el progreso de la sesión de rehabilitación, tal como se observa en la Fig. \ref{fig:AppV1_Terapia} para el movimiento de flexión-extensión.
\begin{figure}[h!]
	\centering
	\includegraphics[width=0.6\textwidth]{figure/pantallas_interfaz/App Ver1.0/Terapia.png}
	\caption{Pantalla de sesión de rehabilitación en curso.\label{fig:AppV1_Terapia}}
\end{figure}

En cualquier momento que el usuario lo requiera durante el proceso de rehabilitación podrá detener el sistema a través del botón  \textit{DETENER RUTINA}, o bien, al finalizar la rutina programada poder volver al menú lo cual mostrará la pantalla de la Fig. \ref{fig:AppV1_Mover_home}.
\begin{figure}[h!]
	\centering
	\includegraphics[width=0.6\textwidth]{figure/pantallas_interfaz/App Ver1.0/Mover_home.png}
	\caption{Pantalla de retorno a menú principal.\label{fig:AppV1_Mover_home}}
\end{figure}
